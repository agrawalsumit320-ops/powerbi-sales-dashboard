# Sales Dashboard - Company XYZ (2020-22)

## 📊 Project Overview
Yeh ek **Power BI Dashboard** hai jo 'Company XYZ' ke 2020 se 2022 tak ke sales data ko analyze karta hai. Iska maqsad sales trends, product performance, aur brand-wise distribution ko visualize karna hai.

## 🚀 Key Insights
Dashboard mein niche di gayi pramukh cheezein shamil hain:
*   **Total Sales:** 99M
*   **Total Cost:** 76M
*   **Total Quantity Sold:** 13K
*   **Top Brand:** Acer

## 📈 Visualizations Included
1.  **Total Quantity by Category:** Bar chart jo dikhata hai ki Monitor, Mouse, aur CPU jaise categories ki kitni demand hai.
2.  **Total Sales by Brand:** Donut chart jo alag-alag brands (Samsung, Dell, Acer, etc.) ke sales share ko darshata hai.
3.  **Sum of Total Sales by Brand:** Pie chart jo overall revenue contribution ko breakdown karta hai.

## 🛠️ Tools Used
*   **Power BI Desktop**: Data visualization aur dashboard creation ke liye.
*   **DAX (Data Analysis Expressions)**: Calculations aur measures banane ke liye.
*   **Power Query**: Data cleaning aur transformation ke liye.

## 📁 How to Use
1.  Is repository ko clone karein.
2.  `.pbix` file ko Power BI Desktop mein open karein.
3.  Data filters ka use karke alag-alag categories aur brands ko explore karein.
